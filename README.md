## :computer: O projeto 
- Foi dado a nós o trabalho de criar funcionalidades para que a agência consiga adicionar, editar e deletar serviços da lista.<br>
- Todos os serviços precisam ser armazenados em uma array ou em um arquivo JSON.
- A página precisa ser renderizada em react

## :dart: Objetivo 
Praticar as habilidades desenvolvidas durante este mês:
   - Lógica de Programação
   - Javascript
   - React

## :mag_right: Para rodar o projeto:
```bash
# Pré requisitos
- Git e Node instalados na máquina

# Faça o clone do repositório
$ git clone <https://github.com/VictorCstr/desafioCRUDjs-gama.git>

# Acesse a pasta do projeto no terminal
$ cd agencia-cronos-adm

# Faça a instalação  das dependências
$ npm install

# Execute a aplicação
$ npm start
```
